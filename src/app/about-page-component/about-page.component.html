About

Welcome! We're glad you made it to our SimCity BUILDITTM calculator.
On this site you'll find interactive tables that allow you to see how long it'll take to make the things you need.
For those technical users out there the source code of this site is available on GitHub for viewing. This site is written in Angular, hosted on Heroku, and deployed via CircleCI.
